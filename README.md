# flipperspambot
The Flipper Zero BadUSB Script automatically presses Ctrl + V and Enter on spamma-script and Command + V and enter on macspamma.

To use:
Open qFlipper
go to the Files section
enter the BadUSB Folder
Drag the Two files into the Folder
On your Flipper:
Go to BadUSB  
Select the File you need
Plug Flipper into target
Press Play on your Flipper
